Project Description

This project aims to design a water analyzer that uses Arduino technology to measure key water quality indicators, such as temperature, pH, and total dissolved solids (TDS). This device provides a rapid and precise response on the potability of water, which is crucial for public health and the environment.

With its intuitive user interface, which includes a computer screen display and color coding system, the analyzer simplifies water quality monitoring and makes it accessible to a wider audience. Testing to date has shown the analyzer to be reliable and compliant with regulatory agency standards.

The implementation of this project has the potential to significantly transform the way individuals and communities assess the quality of their water, providing a valuable tool for environmental education and awareness. The continuous development of this prototype has a target to improve functionality and consider mass production to facilitate distribution and accessibility. This project promises not only to improve public health standards but also to serve as an educational resource for future generations.

Project Concept and Demo

Arduino Coding

In order for the Arduino to be able to interact between the sensors and the computer, we had to do the coding on the Arduino integrated development environment. In summary, the code manipulates the sensor values ​​in order to have useful data. The latter will then be compared to ranges of values ​​that we consider pleasant to drink. For our project, we want to have a temperature between 10 and 25 degrees Celsius, a pH level between 6.5 and 8.5, and to have total dissolved solids between 50 and 250. The temperature data was processed directly using the DallasTemperature library and displays the value in celsius. For the others, we had to use a buffer of a set of values ​​to get an average. And this average is converted from analog to millivolt and then calibrated to have useful values. Depending on the data validations, LED lights will turn on and if they are all green, a blue LED light will turn on and activate a buzzer. In order to be able to further manipulate the data on the website, a number from 0 to 3 is added in series with the sensor values.

Connecting wires to Arduino UNO

Instead of getting lost in the wiring, we used Tinkercad to make virtually all the connections. 200Ω resistors are placed in series with the LEDs and buzzer so they don't burn out.

Coding the website for display

Initially, we intended to bought an LCD but we no longer had a PIN INPUT on the arduino UNO unless we bought another one. But to reduce the cost, we decided to display it on a website. For this, we used VScode as a code editor and the HTML, CSS and Javascript programming language.

Connection Arduino and the website

For the connection between the Arduino and the website, we used the serial port. The data is sent in series and the website receives it and displays it in real time. The data is then processed and displayed in a more user-friendly way. We used node.js to listen to the IDE's serial monitor and pass it into a Javascript file. And using HTML, CSS and this Javascript file, the values ​​are displayed on the website 'localhost:3000'.
